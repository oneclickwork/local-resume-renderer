## 项目经历

::左右 **企业知识库 RAG 系统** || **2024.03-2024.08**

**项目简介：** 面向企业内部文档构建检索增强问答系统，支持 PDF、Markdown、Word 和网页内容入库，覆盖制度查询、产品知识与技术文档问答场景。

**主要工作：**

**1. 文档解析与切分**

- 使用 Python、FastAPI 和 PostgreSQL 构建文档处理服务，实现解析、清洗、切分、向量化与索引更新流程。
- 基于标题层级和句子边界设计结构化切分策略，保留来源页码与章节信息，降低关键信息被截断的概率。

**2. 混合检索与重排**

- 结合向量检索与 BM25 关键词检索，使用 RRF 融合结果，并通过重排模型优化 Top-K 相关性。
- 建立离线评测集，持续跟踪 Recall@5、MRR 和引用准确率，检索召回率由 **81% 提升至 93%**。

**3. 工程化与可观测性**

- 增加任务状态持久化、失败重试和结构化日志，支持批量导入过程中断恢复与问题定位。
- 使用 Playwright 完成端到端测试，覆盖文档上传、检索问答、引用溯源和权限校验等核心流程。

**项目成果：** 累计处理 **2000+** 份文档，核心问答平均响应时间控制在 3 秒以内。
